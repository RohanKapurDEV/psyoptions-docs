領取開發網代幣 - 步驟#

免責聲明所有代幣的價值均為 0 ，這些僅用於測試目的

  1. 確保你的錢包裡有一些 SOL（即將推出更具體說明的文件）

  2. 找到水龍頭 頁面

  3. 連接你最喜歡的 Solana 錢包！

  4. 選擇要空投的代幣

  5. 批准空投交易

  6. 獲得代幣 🤑 🤑 🤑（免責聲明所有代幣的價值均為 0，這些僅用於測試目的）
